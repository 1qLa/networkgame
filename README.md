「ルータ＝関所」「スイッチ＝賢いリーダー」<br>
誰もが知る桃太郎の物語で、目に見えないネットワークの仕組みを直感的に学ぶ3Dインフラ学習ゲーム。

難解なネットワークの基礎概念（IPアドレス、ルータ、スイッチ等）を、3D空間での体験を通じて<br>
直感的に理解させることを目的とした教育用Webアプリケーションです。

---

## 🌟 特徴とこだわり

- **物語と技術の融合**: 「データを目的地に届ける」パケットの旅を、桃太郎の鬼退治の道中に投影。
- **直感的な3Dギミック**: Three.jsを用いた3D空間。家や海を突き抜けない正確な衝突判定を独自実装。
- **徹底した学習UX**: 解答フォームへの自動フォーカスや、Enterキーのみでテンポよく進める操作系を構築。
- **軽量動作**: Webブラウザさえあれば、PCでもスマホでもインストールの必要なく、その場ですぐに遊べます。

## 🛠 技術スタック

### フロントエンド / 3Dグラフィックス
- **Framework**: Vue.js 3
- **3D Engine**: Three.js
- **Styling**: HTML / CSS / JavaScript
- **Modeling**: Blender

---

## 🚀 開発ストーリーと技術的工夫

### 1. 3D空間における「衝突判定」の最適化
手探りでのThree.js実装の中で、キャラクターがオブジェクトを突き抜けないための正確な「衝突判定ロジックの構築」に最も苦労しました。<br>
描画ループ（animate関数）とVueの状態管理が競合しないよう、座標データやベクトル演算のデータフローを<br>
ログ出力で徹底的に可視化・整理することで、ブラウザ上での軽量かつ安定した動作を実現しました。

### 2. コンポーネントの疎結合設計
機能拡張を見据え、ゲームロジックとUI表示の責務を明確に分離するリファクタリングを行いました。<br>
これにより、今後のステージ追加にも柔軟に対応できる保守性の高い設計を担保しています。
